Tutorial de Entornos Virtuales en Python

Un entorno virtual en Python es una herramienta que ayuda a gestionar dependencias y versiones de paquetes para proyectos aislados, evitando conflictos entre librerías que puedan surgir cuando trabajas en diferentes proyectos en la misma máquina.

1. Instalación de venv

La biblioteca estándar de Python incluye venv desde la versión 3.3 en adelante, por lo que no necesitas instalar nada adicional.

# Crear un entorno virtual
python -m venv nombre_del_entorno

# Activar el entorno virtual (Linux/MacOS)
source nombre_del_entorno/bin/activate

# Activar el entorno virtual (Windows)
nombre_del_entorno\Script\activate

2. Instalación de paquetes dentro del entorno virtual

Una vez activado el entorno, puedes instalar los paquetes de Python como de costumbre:

pip install nombre_del_paquete

3. Ver paquetes instalados

Para ver qué paquetes están instalados en el entorno virtual:

pip list

4. Guardar y cargar dependencias

Para exportar las dependencias instaladas en el entorno a un archivo requirements.txt:

pip freeze > requirements.txt

Luego, para instalar las mismas dependencias en otro entorno, puedes usar:

pip install -r requirements.txt

5. Desactivar el entorno virtual

Cuando termines de trabajar en el proyecto, puedes desactivar el entorno virtual con:

deactivate

6. Eliminar un entorno virtual

Para eliminar un entorno virtual, simplemente borra la carpeta del entorno virtual (ya que es independiente):

rm -rf nombre_del_entorno

7. Ventajas de los entornos virtuales

  • Aislamiento: Cada proyecto tiene sus propias dependencias, lo que evita conflictos de versiones.
  • Reproducibilidad: Puedes compartir tu proyecto junto con el archivo requirements.txt, lo que permite a otros instalar las mismas dependencias de forma sencilla.

8. Ejemplo Completo

# Crear un entorno virtual
python -m venv mi_entorno

# Activar el entorno virtual
source mi_entorno/bin/activate  # o mi_entorno\Scripts\activate en Windows

# Instalar paquetes
pip install requests flask

# Ver paquetes instalados
pip list

# Guardar dependencias en requirements.txt
pip freeze > requirements.txt

# Desactivar el entorno virtual
deactivate
