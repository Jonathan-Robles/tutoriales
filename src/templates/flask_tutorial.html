Mini Tutorial de Flask

Paso 1: Configura tu entorno

1. Crea un entorno virtual:

python -m venv venv

2. Activa el entorno virtual:

venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS/Linux

3. Instala Flask:

pip install Flask

Paso 2: Crea la estructura del proyecto

Crea un archivo llamado app.py en la raíz de tu proyecto:

/mi_proyecto
├── venv/
└── app.py

Paso 3: Escribe tu aplicación Flask

Abre app.py y escribe el siguiente código:

Paso 4: Ejecuta tu aplicación

En la terminal, asegúrate de que estás en la carpeta del proyecto y ejecuta:

python app.py

Paso 5: Prueba tu aplicación

1. Abre tu navegador y ve a http://127.0.0.1:5000/ para ver el mensaje de bienvenida.

2. Ve a http://127.0.0.1:5000/about para ver la página "Acerca de".

3. Dirígete a http://127.0.0.1:5000/form para enviar un nombre a través del formulario. Después de enviarlo, serás redirigido a una página de saludo personalizada.

Pasos Siguientes

Este es solo un tutorial básico. A partir de aquí, puedes:

  • Agregar una base de datos usando Flask-SQLAlchemy.
  • Manejar autenticación con Flask-Login.
  • Usar Flask-WTF para formularios más complejos.
  • Implementar rutas y controladores más avanzados.
